# House Price Prediction Using Machine Learning

## Project Overview
This project predicts house prices using Machine Learning techniques. The system compares multiple regression models including Linear Regression, Random Forest, and XGBoost to identify the best-performing model for accurate house price prediction.

## Features
- House Price Prediction
- Linear Regression Model
- Random Forest Model
- XGBoost Model
- Model Evaluation
- Overfitting Analysis
- Streamlit Web Application Deployment

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit

## Dataset
- kc_house_data.csv

## How to Run:
### Step 1: Install Dependencies
pip install pandas numpy scikit-learn xgboost streamlit

### Step 2: Train the Model
py train_models.py
This will generate:
house_price_model.pkl

### Step 3: Run the Application
py -m streamlit run app.py
